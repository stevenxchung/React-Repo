Note: You must have the most recent version of Node.js installed to run.

Steps to run:

1) Copy and paste the following in your command window:

npm install -g create-react-app
create-react-app my-app

2) Delete all files in the src/ folder of the new project:

cd my-app
rm -f src/*

3) Copy and paste the src/ folder and contents into your workspace.

4) Type the following into the command window:

npm start

5) To optimize build:

npm run build
